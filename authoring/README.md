# Authoring workspace

This directory is reserved for the future Obsidian author workspace.

During TASK-000:

- do not create story content;
- do not create named character, location, faction, item, or scene files;
- do not define a runtime content schema;
- do not configure Obsidian plugins.

The directory exists only to make the long-term boundary visible:

- author material lives here;
- the game does not run inside Obsidian;
- future publishable content must pass an explicit validation and compilation pipeline.
